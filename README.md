# 🏥 Hospital Management System

نظام إدارة مستشفيات متكامل مبني باستخدام **Node.js** + **Express** + **TypeScript** + **PostgreSQL** + **Prisma**.

## 📋 المتطلبات

- **Node.js** (إصدار 18 أو أحدث)
- **PostgreSQL** (إصدار 14 أو أحدث)
- **npm** أو **yarn**
